# ML Bootcamp Python

Repozytorium szkoleniowe tworzone podczas przerabiania kursu **Machine Learning Bootcamp w języku Python cz. I — od A do Z**.

Celem projektu jest praktyczne poznanie podstaw uczenia maszynowego w Pythonie, ze szczególnym naciskiem na uczenie nadzorowane, przygotowanie danych, trenowanie modeli oraz ocenę ich skuteczności.

## Zakres nauki

Repozytorium obejmuje ćwiczenia, notatki i przykłady związane z klasycznym Machine Learningiem:

* podstawy uczenia maszynowego,
* uczenie nadzorowane,
* regresja,
* klasyfikacja,
* eksploracyjna analiza danych,
* przygotowanie danych do modelu,
* inżynieria cech,
* podział danych na zbiór treningowy, walidacyjny i testowy,
* trenowanie modeli,
* walidacja modeli,
* ocena skuteczności,
* optymalizacja modeli,
* przeuczenie i niedouczenie.

## Algorytmy

W ramach kursu omawiane i ćwiczone są między innymi:

* regresja liniowa,
* regresja wielomianowa,
* regresja logistyczna,
* K-nearest neighbors,
* drzewa decyzyjne,
* Random Forest,
* Support Vector Machine,
* Naive Bayes.

## Główne technologie

* Python
* pandas
* NumPy

## Cel repozytorium

To repozytorium nie jest gotową biblioteką ani projektem produkcyjnym.
Służy do nauki, eksperymentowania i utrwalania wiedzy z zakresu Machine Learningu.

Najważniejsze cele:

1. Zrozumieć pełny przepływ pracy przy budowaniu modeli ML.
2. Nauczyć się przygotowywać dane do trenowania modeli.
3. Przećwiczyć podstawowe algorytmy regresji i klasyfikacji.
4. Poznać sposoby oceny jakości modeli.
5. Zrozumieć problemy przeuczenia i niedouczenia.
6. Utrwalać wiedzę w kodzie, dokumentacji i quizach.

## Przykładowy workflow ML

Typowy proces pracy nad modelem obejmuje:

1. Wczytanie danych.
2. Eksploracyjną analizę danych.
3. Czyszczenie i przygotowanie danych.
4. Podział na zbiory treningowe, walidacyjne i testowe.
5. Wybór algorytmu.
6. Trenowanie modelu.
7. Ocenę skuteczności.
8. Optymalizację parametrów.
9. Interpretację wyników.
